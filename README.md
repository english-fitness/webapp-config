# webapp-config
Config files for webapp.
1. Rename the folder to config after clone.
2. Put the folder into webapp/protected/
It should look like this
| webapp
|    |- ...
|    |- protected
|    |   |- ...
|    |   |- config
|    |   |    |- ...
|    |   |    |- main.default.php
|    |   |    |- ...
|    |   |- ...
|    |- ...
